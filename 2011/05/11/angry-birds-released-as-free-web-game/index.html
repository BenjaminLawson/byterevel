    Angry Birds Released As Free Web Game

    less than 1 minute read

    Angry Birds, the extremely popular physics game where you launch wingless birds at green pigs, has been released by Rovio Mobile (the developers of the game) on their website. The game is available as a free WebGL game , and it offers both standard definition and high definition graphics. It was intended as an application on the Google Chrome Web Store, but it also works on many other browsers.

    Check it out at: http://chrome.angrybirds.com/
